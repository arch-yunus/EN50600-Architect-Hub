# Contributing to EN50600-Architect-Hub

First off, thank you for considering contributing to this authoritative resource for EN 50600 standards. It's people like you that make the engineering community a better place.

## Code of Conduct

This project and everyone participating in it is governed by the [EN50600-Architect-Hub Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, open a new issue with a clear title and description.

### Suggesting Enhancements
* Open an issue with the [Enhancement] tag.
* Describe the proposed change and its benefits to the EN 50600 architectural framework.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added documentation, ensure it follows the premium technical tone and formatting of the project.
3. Submit a pull request with a detailed description of your changes.

## Style Guide
* Use clear, technical, and professional language.
* Maintain the academic structure of the documentation.
* Ensure all technical terms follow the CENELEC EN 50600 nomenclature.

Thank you for your contribution!
